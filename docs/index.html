Features
Everything you need to
keep your agents in check
◉

Live Session Monitoring

See every Claude Code and Codex session running on your machine. Status, current tool, elapsed time — updated in real-time via hooks.

⚠

Permission Routing

Intercepts permission prompts and shows them in a native banner. Allow, deny, or allow-always without switching to the terminal.

❓

Question Answering

When Claude asks a question, see it in the notch panel with clickable options. Answer directly from the UI.

⏱

Usage Tracking

Monitor your 5-hour and weekly rate limits for both Claude and Codex. See remaining percentages at a glance with reset time estimates.

▶

Terminal Jump

Click any session to jump directly to its terminal window. Supports Warp, Ghostty, iTerm2, VS Code, and Terminal.app.

♫

Sound Notifications

Customizable sound effects for permissions, questions, and completions. Mario, Pop, or Chime presets — or import your own.

How it works
Three steps to
agent awareness
1

Install & Launch

Download the .dmg, drag to Applications, and launch. Coder Island appears in your macOS notch area and starts scanning for sessions.

→
2

Enable Hooks

Toggle hooks in Settings. Lightweight scripts relay Claude Code events to the app via a Unix socket — zero latency.

→
3

Code with Confidence

Run your agents as usual. Coder Island surfaces permissions, questions, and status in real-time without context-switching.

Settings
Fully configurable
to your workflow
Hook Integration
Install lightweight hook scripts so permission prompts and questions appear in the app instead of blocking the terminal. Toggle on in Settings → Interaction.
Smart Suppression
Don't auto-expand the panel when the agent's terminal is already in focus. Prevents the notch from interrupting what you're looking at.
Sound Presets
Mario, Pop, or Chime built-in profiles. Toggle per-event (permissions, questions, completions, app start) or import your own audio files.
Usage Display
Show 5-hour and weekly rate limit percentages inline next to agent icons. Hover for reset times. Switch between compact and detailed modes.
Launch at Login
Start automatically after login. Grant Accessibility permission once for terminal tab switching and foreground app detection.
Multi-Display
Choose which screen the notch lives on. Auto-hide in fullscreen mode. Works across all Spaces.
